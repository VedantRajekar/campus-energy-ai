## 🆘 Troubleshooting

### "Nothing happens when I open HTML"
- Make sure you're using a modern browser (Chrome, Firefox, Safari, Edge)
- Check that JavaScript is enabled
- Try right-click → Open With → Chrome

### "Python not found"
- Install from: https://www.python.org/downloads/
- Check "Add to PATH" during installation

### "Charts not loading"
- Wait a few seconds for Chart.js to load from CDN
- Check internet connection (needed for Chart.js library)

### "Want to modify the data?"
- Edit the Python file
- Change parameters like `days=14` to `days=30`
- Run the script again to regenerate data
